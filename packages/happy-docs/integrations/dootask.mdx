[DooTask](https://www.dootask.com/) is an open-source project management tool. Happy integrates with DooTask to let you manage tasks and launch AI coding sessions directly from your task list.

## Connecting to DooTask

1. Open **Settings** in the Happy app
2. Navigate to the DooTask section
3. Enter your DooTask server URL and authenticate

## Features

Once connected, you can:

- **View tasks** — Browse your DooTask projects and task lists
- **Create tasks** — Add new tasks from the Happy app
- **Real-time sync** — Task updates are synced in real time via WebSocket
- **Launch AI sessions** — Tap a task to start an AI coding session with the task description as context
- **Reactions** — Add emoji reactions to tasks

## Task-driven development

The DooTask integration enables a workflow where you:

1. Pick a task from your project board
2. Launch an AI session directly from the task
3. The AI agent receives the task description as context
4. Complete the work and update the task status

This bridges project management and AI-assisted development in a single app.
