---
title: "分享"
description: "安全地与队友分享 AI 编程会话。"
---

Happy 支持与其他 Happy 用户分享会话，可以通过好友邀请进行私密分享，也可以通过公开链接分享。

## 好友

在私密分享会话之前，您需要先添加好友：

1. 进入应用中的**好友**标签页
2. 点击**搜索**通过用户名查找用户
3. 发送好友请求 — 对方会在待处理请求中看到
4. 对方接受后，即可互相分享会话

<Info>
好友发现功能需要 **GitHub 账号**。在**设置 > GitHub** 中关联您的 GitHub 账号，以获得可被他人搜索的用户名。
</Info>

## 私密分享

直接与好友分享会话：

1. 打开要分享的会话
2. 点击顶部栏的分享图标
3. 从列表中选择好友
4. 选择访问级别：
   - **查看** — 可以阅读消息，但不能发送
   - **编辑** — 可以阅读和发送消息
   - **管理** — 拥有完全控制权限，包括与他人分享

分享的会话采用端到端加密。您的好友通过安全的 NaCl Box 密钥交换接收会话加密密钥 — 服务器无法读取分享的内容。

分享的会话实时同步：消息、Git 状态和语音聊天会广播给所有参与者。

## 公开分享链接

创建一个任何人都可以用来查看会话的链接：

1. 打开会话
2. 点击分享 > **创建公开链接**
3. 配置选项：
   - **有效期** — 链接的有效时长
   - **使用次数限制** — 最大查看人数
4. 复制并发送链接

公开链接使用基于令牌的加密密钥 — 链接本身包含解密密钥，因此任何拥有链接的人都可以查看会话。

<Warning>
公开链接的 URL 中包含解密密钥。任何获得链接的人都可以读取会话内容。请谨慎分享。
</Warning>

## 管理分享

查看和管理会话的所有分享：
- 撤销单个用户的访问权限
- 删除公开链接
- 更改访问级别
