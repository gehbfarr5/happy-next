产物查看器可让您直接在 Happy 应用中浏览项目的代码和文件。

## 代码浏览器

浏览项目的文件树并查看带有语法高亮的源代码。支持的功能包括：

- **文件导航** — 浏览目录和文件
- **语法高亮** — 识别语言的代码着色
- **搜索** — 按名称查找文件

## Git 集成

查看项目的 Git 历史记录：

- **提交日志** — 浏览近期提交
- **分支选择器** — 在分支之间切换
- **差异查看器** — 查看每次提交的变更，支持并排对比和内联差异

## 文件类型

产物查看器支持以下类型：

- **源代码** — 支持所有主流语言的语法高亮
- **图片** — 预览 PNG、JPG、SVG 及其他图片格式
- **Markdown** — 带格式的渲染展示
- **差异** — 内联和并排对比视图
