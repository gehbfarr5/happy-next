## 与你的机器配对

<Steps>
  <Step title="在电脑上启动 Happy 会话">
    在终端中运行 `happy`。将会显示一个二维码。
  </Step>
  <Step title="打开 Happy 应用">
    在手机上启动应用。首次启动时，系统会引导你完成账户设置。
  </Step>
  <Step title="扫描二维码">
    点击主页上的扫描图标，将摄像头对准终端中的二维码。
  </Step>
  <Step title="连接完成">
    你的手机已完成配对。你可以在应用中看到当前活跃的会话。
  </Step>
</Steps>

## 账户

Happy 使用**加密认证**——无需密码。你的身份是设备上生成的密钥对。当你扫描二维码时，手机和电脑会交换公钥以建立安全连接。

## 设置

点击齿轮图标进入设置。主要选项：

- **外观** — 浅色、深色或自动主题
- **语音** — 语音提供方和语言设置
- **通知** — 推送通知偏好
- **AI 后端配置** — 为 Claude、Codex 或 Gemini 配置自定义 API 端点
- **GitHub** — 关联你的 GitHub 账户以使用社交功能
