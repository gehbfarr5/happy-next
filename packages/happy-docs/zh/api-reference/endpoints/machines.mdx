---
title: "机器"
description: "注册和管理已连接的设备。"
---

## 创建机器

```
POST /v1/machines
```

注册一台新机器（运行 CLI 的计算机）。

**请求体：**

| 字段 | 类型 | 说明 |
|-------|------|-------------|
| `metadata` | string | 加密的机器元数据（主机名、操作系统等） |
| `encryptionKey` | string | 加密的机器密钥 |

## 列出机器

```
GET /v1/machines
```

列出已认证用户的所有机器。

## 更新机器

```
PATCH /v1/machines/:id
```

更新机器元数据或守护进程状态。

**请求体：**

| 字段 | 类型 | 说明 |
|-------|------|-------------|
| `metadata` | string | 加密的机器元数据 |
| `daemonState` | string | 加密的守护进程状态 |
| `activity` | object | 活动跟踪数据 |
