title: "Artifacts"

---

The Artifacts viewer lets you browse your project's code and files directly from the Happy app.

## Code browser

Navigate your project's file tree and view source code with syntax highlighting. Supported features:

- **File navigation** — Browse directories and files
- **Syntax highlighting** — Language-aware code coloring
- **Search** — Find files by name

## Git integration

View your project's Git history:

- **Commit log** — Browse recent commits
- **Branch selector** — Switch between branches
- **Diff viewer** — See what changed in each commit with side-by-side or inline diffs

## File types

The Artifacts viewer supports:

- **Source code** — Syntax highlighted for all major languages
- **Images** — Preview PNG, JPG, SVG, and other image formats
- **Markdown** — Rendered with formatting
- **Diffs** — Inline and side-by-side comparison views
